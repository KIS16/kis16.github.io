# kis의 깃 블로그

Bootstrap 템플릿으로 제작된 개인 블로그 웹사이트입니다.

## 프로젝트 구조

```
kis16.github.io/
├── .git/                  # Git 버전 관리 디렉토리
├── assets/               # 정적 파일 디렉토리
│   ├── img/             # 이미지 파일
│   └── favicon.ico      # 웹사이트 파비콘
├── css/                  # 스타일시트 디렉토리
│   └── styles.css       # 메인 스타일시트
├── js/                   # 자바스크립트 디렉토리
│   └── scripts.js       # 메인 자바스크립트 파일
├── index.html           # 메인 페이지 (블로그 포스트 목록)
├── about.html           # 소개 페이지
├── post.html            # 블로그 포스트 상세 페이지
├── contact.html         # 연락처 페이지
└── README.md            # 프로젝트 문서
```

## 설명

이 웹사이트는 Bootstrap 템플릿을 사용하여 제작된 개인 블로그입니다. 주요 기능은 다음과 같습니다:

- Bootstrap 프레임워크를 활용한 반응형 디자인
- 깔끔하고 모던한 UI
- 블로그 포스트 목록 및 상세 페이지
- 소개 및 연락처 페이지
- 소셜 미디어 통합
- Font Awesome 아이콘
- Google Fonts 통합

## 페이지 구성

- **홈 (index.html)**: 블로그 포스트 목록을 보여주는 메인 페이지
- **소개 (about.html)**: 블로그 작성자에 대한 정보
- **포스트 (post.html)**: 개별 블로그 포스트 템플릿
- **연락처 (contact.html)**: 연락처 정보 및 양식

## 사용 기술

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- Google Fonts
