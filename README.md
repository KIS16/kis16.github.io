# KIS16 · 개발·연구 포트폴리오

박사과정 포트폴리오용 GitHub Pages 사이트입니다.  

- **About**: 소개·연락처  
- **Projects**: 참여/주도 프로젝트·연구 결과물  
- **Posts**: 기술 블로그·연구 노트 

## 콘텐츠 작성

- **포스트**: `_posts/`에 `YYYY-MM-DD-제목.md` 추가. 또는 `bundle exec jekyll post "제목"` 실행 후 내용 작성.
- **프로젝트**: `_projects/`에 `프로젝트명.md` 추가. `_projects/sample-project.md`를 참고해 front matter와 본문을 채우면 됩니다.

## 기타

[Jekyll](https://jekyllrb.com/) + [Hydejack](https://hydejack.com/) 테마로 빌드됩니다.
